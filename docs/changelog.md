- シーケンス一覧の各行に […] メニューボタンを追加しました。名前の変更・削除がその場でできます（行の右クリックでも同じメニューを表示）。

- ライブラリの左サイドバーのフォルダツリーを、フォルダ左の矢印（chevron）クリックで開閉できるようにしました。折りたたみ状態はリロード・再起動後も保持されます。
- 動画生成パネルで、ワークフロー選択を動画プロンプトの下に移動しました。

### 2026-07-23

- 上部バー右端に設定ボタン（歯車アイコン）を追加し、設定パネルにメンテナンス系の操作をまとめました。「ライブラリの保存先の変更・既定に戻す」「インデックス再構築」「Embedding 更新」はこのパネルから実行します（上部バーの Embedding 更新・インデックス再構築ボタンは設定パネルへ移動）。パネルは ✕ ボタンまたは外側クリックで閉じます。
- ウインドウの既定サイズを、コンテンツ領域が 1920x1080 になるように変更しました。
- アプリ内のアイコンを絵文字からフラットデザインの SVG アイコン（線画・単色）に置き換えました。ツールバー、右クリックメニュー、ボタン、フォルダツリー、動画バッジ（🎞 NEW → フィルムアイコン）、生成キュー、シーケンスの再生コントロール・BGM パネル、スニペット一覧などが対象です。色はテキスト色に追従するため、状態色（NEW バッジの琥珀色など）もそのまま引き継がれます。
- 上部バーの Forge / ComfyUI / LLM / Embedding の状態表示を、旧 Image Assistant のように遷移がわかるようにしました。「停止（灰）/ 起動中・ロード中（琥珀・点滅）/ 起動済み（緑）/ エラー（赤）」の 4 色で表示し、起動中は「起動中…」「モデルをロード中…」「セットアップ中…」のテキストも表示します。遷移中は状態の確認間隔を自動的に短縮（8 秒 → 1.5 秒）するため、起動完了がすぐ反映されます。
- ライブラリのフォルダ名を大文字から小文字（またはその逆）へ変更できない問題を修正しました。Windows のファイルシステムが大文字小文字を区別しないため「同名フォルダが既に存在する」と誤判定されていました。

### 2026-07-22

- スニペット編集のファイル一覧を右クリックするとメニューを表示し、ファイル名の変更（サブフォルダへの移動も可）と削除ができるようにしました。
- スニペット編集で、項目一覧の項目を左のファイル一覧へドラッグ＆ドロップすると、その項目を別ファイルへ移動できるようにしました（移動先へ追記・元ファイルから削除して両方保存。JSON 直接編集中は無効）。
- シーケンス編集のノードを右クリックするとメニューを表示し、「ライブラリの元動画を表示」でライブラリタブに切り替えて該当画像・動画を選択できるようにしました。ノード削除もメニューから行えます。
- 動画生成パネルに「生成前に LLM をアンロード（VRAM を空ける）」チェックボックスを追加しました（既定オン、設定は保存されます）。オンの間は、キューからの動画生成の直前に llama-server を停止して VRAM を空けます。
- 複数選択した画像をドラッグして、グリッド内の並べ替えもまとめてできるようにしました（これまで複数ドラッグはフォルダ移動のみ）。選択した画像は元の並び順を保ったままドロップ位置に挿入されます。
- 動画生成パネルを開いたまま生成した動画を動画一覧でクリックしたとき、右パネルがその動画のプロパティに切り替わらない問題を修正しました。
- 既存画像から「✨ この設定で新規生成」「🖼 新規生成でキューに追加」で画像を生成したとき、生成結果を元の画像のすぐ左隣（一覧は新しいものほど左・上のため直前）に並べるようにしました。通常のフォルダ生成は今まで通り一覧の先頭に入ります。
- 新しく生成された画像のカードに NEW バッジを表示するようにしました。カードをクリックすると消えます（リロード・再起動後も未クリックなら表示されたままです）。
- 新しく生成された動画にも、動画一覧（下部ストリップ）のカードに NEW バッジを表示するようにしました。画像と同じくクリックで消えます。
- 未確認の新規動画がある画像のグリッドカードにも「🎞 NEW」を表示するようにしました。動画由来の NEW は琥珀色、画像自体の NEW は青で区別されます。動画をクリックして確認すると画像側の表示も消えます。
- スニペット機能を移植しました。VSCode `.code-snippets` 形式のファイルを `snippets/` フォルダ（変更可）から読み込み、プロンプト作成に利用できます。
- Prompt / Negative Prompt / 動画プロンプトの入力中に、prefix にマッチするスニペットの自動候補を表示するようにしました（↑↓ で選択、Tab / Enter で挿入、Esc で閉じる）。
- スニペットタブをフォーム式の編集 UI にしました。ファイル一覧・項目一覧（全ファイル横断検索つき）・フォーム（Name / Prefix / Description / Body）で編集でき、生 JSON の直接編集にも切り替えられます。
- シーケンス編集とスニペット編集で Ctrl+S 保存に対応しました。
